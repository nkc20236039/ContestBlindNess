# インタラクトガイド

![参考画像：インタラクトガイド](/Specifications/images/Player/ActionGuide.png)

## 基本挙動
1. オブジェクトがインタラクト可能であることを示すために、ガイドを表示する。  
2. 視点のあっているオブジェクトがインタラクト可能な場合、以下の挙動を行う。
- 照準の右上当たりに小さく**インタラクトキーを表示**
- **照準の色を緑**に変更
